## Local Development
#### Using Vite Hot Module Reloading
```bash
$ dfx start --background --clean
$ dfx deploy shrine_backend
$ npm run dev
$ dfx stop
```

#### Testing Prod on Local
```bash
$ npm run build
$ dfx deploy
```


## Deploying to Mainnet
```bash
$ npm run build
$ dfx deploy --network ic
```
